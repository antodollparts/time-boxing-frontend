# Time Boxing - Frontend
Time Boxing es una aplicación diseñada para ayudar a los usuarios a organizar su día mediante la técnica de time boxing, que consiste en asignar bloques de tiempo específicos a cada tarea.

## Objetivos

- Aplicar la técnica de time boxing para la gestión del tiempo.
- Permitir la creación y edición de tareas.
- Visualizar bloques de tiempo en una agenda diaria.
- Mejorar la productividad y el enfoque del usuario.
